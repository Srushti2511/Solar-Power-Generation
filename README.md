# ☀️ Solar Power Generation Prediction using Machine Learning

## 📌 Project Overview

Solar energy is one of the fastest-growing renewable energy sources. Accurate prediction of solar power generation helps improve energy management, reduce operational costs, and support efficient power grid planning.

This project uses Machine Learning techniques to predict solar power generation based on environmental and weather-related parameters. The model analyzes historical data and forecasts the expected power output, helping optimize renewable energy production.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Solar-Power-Generation.git
```

Navigate into the project

```bash
cd Solar-Power-Generation
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
jupyter notebook
```

or

```bash
python dashboard.py
```
